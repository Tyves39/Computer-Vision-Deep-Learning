# Computer Vision & Deep Learning

This section showcases image processing and classification projects using advanced neural network architectures.

## Projects

* **Image Preprocessing**: Advanced manipulation using OpenCV (Filtering, Sharpening, and Data Augmentation).
* **MNIST Recognition**: Hand-written digit classification using MindSpore with 97.6% accuracy.
* **MobileNetV2**: Transfer Learning for flower species classification optimized for mobile performance.
* **ResNet-50**: Deep Residual Network implementation for high-accuracy image categorization.

## Key Skills
* Frameworks: MindSpore, PyTorch.
* Libraries: OpenCV, Matplotlib, NumPy.
* Techniques: Transfer Learning, Data Augmentation, Model Checkpointing.
